# Shopping List Manager 

A simple command-line Python application that helps users create and manage their shopping list.

This project is designed for beginners to practice Python fundamentals such as functions, loops, file handling, and list operations.

---

## Features

-  Add new items
-  View shopping list
-  Remove items
-  Clear the shopping list
-  Save shopping list to a text file
-  Load shopping list from a file automatically
-  Exit application

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/shopping-list-manager.git
```

Move into the project folder

```bash
cd shopping-list-manager
```

Run the application

```bash
python main.py
```
